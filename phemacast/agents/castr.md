Castr is an agent that can render Phema into documents or media for audiences.
* Castr can register itself on plaza
* Cast is a Practice can be called by agents to perform transformation from phema to other formats.

Castr UI
* select Phema from plaza
* select audience type, preference, language, format, etc.
* generate a specific type of media, such as PDF, PPT, video, audio, etc.
* generate output and store in local pool or plaza
* a config file for each type of process and output
