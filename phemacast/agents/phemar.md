What is Phemar?
* Phemar is an agent that can generate Phema.
* Phemar can get data from Pulses defined in Phema, and use the data to generate more static Phema.


Phemar UI
* not only a single Phema Editor, it should be a complete Phema management system
* can store Phema in local pool or register on plaza. 
* View, search, edit Phemas in local pool. for Phema on the plaza, 
* only owner Phemar can edit or de register.
** available pulses
* list all pulses from plaza
* 2 lines for each pulser, display the number of available active Pulser for this pulse 
* when drag and drop into section, a property modal window to select Pulser, run get data, and select fields returned by pulser to display in the section.
